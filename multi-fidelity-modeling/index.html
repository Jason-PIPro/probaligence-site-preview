What is multi-fidelity modeling?

Multi-fidelity modeling combines data of different cost and accuracy in one model. Cheap, lower-accuracy data is fast and broad; expensive, high-accuracy data is sparse. STOCHOS learns how the levels relate, so cheap data provides coverage and expensive data anchors accuracy.

01

The idea, in plain terms

Imagine a coarse simulation you can run hundreds of times and a precise test you can run only a few times. Used separately, one is broad but rough and the other is accurate but narrow. Multi-fidelity links them: the model leans on the cheap data for shape and the expensive data for correction, giving a better answer than either alone.

02

How STOCHOS does it

STOCHOS learns the relationship between fidelity levels inside one probabilistic model. It uses broad, inexpensive simulations to cover the space and sparse, high-quality tests to correct accuracy, and reports confidence throughout. This is built into the DIM-GP approach, so multi-fidelity data needs no separate, hand-tuned pipeline.

03

When to use it, and what goes in and out

Use multi-fidelity when you have data sources of different cost and trust: coarse and fine simulations, simulation and experiment, or lab and production runs.

Inputs

The datasets and their fidelity levels.

Outputs

Predictions that combine coverage and accuracy, with uncertainty, so high-cost runs go only where they help.

A concrete example

Coarse and fine, combined

In coatings, liquid-state measurements are cheaper than dried-film tests, and lab data is cheaper than production data. Multi-fidelity modeling combines them so the model predicts final properties without running every expensive test, and flags where a high-fidelity run would most reduce uncertainty.

FAQ

Frequently asked questions

Why not just use the high-fidelity data?+

High-fidelity data is accurate but scarce and expensive, so on its own it rarely covers the design space. Multi-fidelity modeling adds cheap, broad data for coverage and uses the limited high-fidelity data to anchor accuracy, which usually gives a better prediction at lower total cost.

Does this only apply to simulation?+

No. The same idea applies wherever data sources differ in cost and accuracy, including simulation versus experiment and lab versus pilot versus production. STOCHOS handles these combinations within one model.
