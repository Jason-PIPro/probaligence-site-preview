FAQ

Frequently asked questions

What does PI Probaligence do?+

PI Probaligence develops local, probabilistic AI for engineering and R&D. It uses your existing simulation, experimental, and process data to build fast surrogate models that predict results for new variants and quantify uncertainty. This shortens development cycles and supports more reliable decisions, without needing an in-house ML team.

What is probabilistic AI?+

Probabilistic AI returns a prediction together with a measure of confidence, not a single number with no context. For engineering, that means you can see where a model is reliable and where it is not, which matters when decisions carry cost or risk and data is limited.

STOCHOS versus STOCHOS Flow?+

STOCHOS is the probabilistic intelligence layer: the modeling, prediction, and optimization engine. STOCHOS Flow is the visual, agentic workflow layer to build, automate, reuse, and export workflows. STOCHOS does the predicting; STOCHOS Flow makes it accessible and repeatable for a team.

Does the software run locally?+

Yes. PI is local-first. Every workflow runs on your own infrastructure, and STOCHOS Flow can use a local language model through Ollama. Optional cloud models are available only if you choose them. Local execution keeps sensitive data and IP on your systems, which matters for confidentiality and compliance.

Can it run offline?+

Yes. STOCHOS can be installed and run fully offline. In that mode it makes no outbound network connections, so the entire workflow stays sealed on your own systems: the predictive engine and a local language model through Ollama run offline, and cloud models are simply off.
