Science / the algorithm

How STOCHOS works.

STOCHOS is built on DIM-GP, a Deep Infinite Mixture of Gaussian Processes. It joins the pattern recognition of deep learning with the calibrated uncertainty of Gaussian processes in one non-stationary, probabilistic model, and returns every prediction with a measure of confidence.

The gap

Why standard models fall short here

Classical Gaussian processes give calibrated uncertainty but struggle with large, high-dimensional, or non-stationary data. Deep networks scale but usually return a single number with no confidence. DIM-GP combines the two.

Classical GP

Calibrated uncertainty, but struggles with large, high-dimensional, or non-stationary data.

Deep networks

Scale to complex, high-dimensional relationships, but usually return one number with no confidence.

DIM-GP

A deep architecture for capacity, a Gaussian-process treatment for uncertainty, reliable when data is scarce.

Uncertainty

Built in, not bolted on

Every DIM-GP prediction carries a confidence interval. STOCHOS uses those bounds directly when it optimizes, so the search accounts for what the model does not yet know. This is what separates reliable interpolation from risky extrapolation.

One family

One model family, many data shapes

DIM-GP is a family, not a single model. The same probabilistic core runs across all of them, for both regression and classification, plus multi-fidelity versions, so a team learns one approach instead of a new tool per problem.

From model to decision

Optimization that uses the uncertainty

STOCHOS pairs DIM-GP with Bayesian optimization that adapts after each result. It handles single and multi-objective goals, constraints, and data sources of different cost, and returns a Pareto front when objectives compete. Noise-robust variants find stable optima when measurements are noisy.

Runs where your data is

FAQ

Frequently asked questions

What is DIM-GP?+

DIM-GP, a Deep Infinite Mixture of Gaussian Processes, is the core STOCHOS algorithm. It combines neural networks and Gaussian processes into a non-stationary, probabilistic model designed for small, noisy datasets, changing geometries and meshes, and multi-fidelity data, without manual hyperparameter tuning.

Why combine deep learning and Gaussian processes?+

Deep learning brings capacity to model complex, high-dimensional relationships. Gaussian processes bring calibrated uncertainty. Engineering needs both: a flexible model that also reports how far to trust each prediction. DIM-GP is built to deliver capacity and uncertainty in one model, tuned for small, expensive datasets.

Can STOCHOS run on-premise?+

Yes. STOCHOS runs on Windows and Linux, on CPU or GPU, and can be installed without an internet connection by providing the packages directly. Core modeling and optimization run on your own infrastructure, so sensitive simulation, test, and formulation data stays under your control.
