Science

The science behind STOCHOS.

STOCHOS is built on probabilistic machine learning made for engineering data: small, noisy, and expensive to produce. Start with the core algorithm, DIM-GP, and the methods built on it: surrogate modeling, uncertainty quantification, Bayesian optimization, multi-fidelity modeling, sensitivity analysis, and generative design.

How STOCHOS works Explore STOCHOS
The algorithm

Start with the algorithm

Everything in STOCHOS rests on one algorithm, DIM-GP, a Deep Infinite Mixture of Gaussian Processes. It joins deep learning with calibrated Gaussian-process uncertainty in a single model built for small, noisy, expensive data, and it returns every prediction with a measure of confidence.

The methods

The methods STOCHOS uses

The methods build on DIM-GP and work together: surrogate modeling for fast prediction, uncertainty quantification to show where to trust it, Bayesian optimization to choose the next run, multi-fidelity modeling to cut cost, and sensitivity analysis to explain what drives the result. Generative design expands the search beyond fixed variants.
