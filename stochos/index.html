The intelligence layer

The probabilistic intelligence layer for physical systems.

STOCHOS learns from simulation, measurement, and test data to predict new variants in seconds, quantify uncertainty on every output, and guide the next decision. It is built for the small, noisy, expensive data that real physical problems produce, and it runs on your own infrastructure.

Why STOCHOS

Use the data you already have, more intelligently

Each full simulation, experiment, or prototype can be expensive. STOCHOS builds probabilistic surrogate models that learn from previous results, evaluate new variants in seconds, and show where another run would add the most value. You spend expensive solves where they matter, not on every candidate.

Predict faster

Replace repeated full evaluations with fast surrogate predictions for new designs, recipes, and process settings.

Decide smarter

Quantify confidence instead of relying on a single value with no indication of risk.

Experiment less

Use Bayesian optimization to choose the next simulation or experiment from your objectives and constraints.

The DIM-GP model
Core algorithm

DIM-GP: deep learning with Bayesian uncertainty

A Deep Infinite Mixture of Gaussian Processes. It combines the pattern recognition of deep learning with the uncertainty estimation of Gaussian processes, in one non-stationary probabilistic model, designed for small, noisy datasets with little manual tuning.

  • Handles small datasets, noisy measurements, and nonlinear relationships.
  • Shows where a prediction can be trusted and where more data would help.
  • Supports changing geometries, meshes, multi-fidelity data, and field outputs.
  • Reduces manual model tuning and runs on local infrastructure.

Supported data

Built for real engineering and R&D data

STOCHOS is not limited to one table format or one kind of output. The same probabilistic approach spans engineering, chemistry, and formulation data, so one method covers problems that usually need several tools.

Scalar and tabular

Material properties, process parameters, recipes, test results.

Signals and time series

Vibration curves, force-displacement, spectra, process histories.

2D and 3D fields

CFD pressure fields, FEM stress maps, temperature distributions.

Geometries and volumes

STL geometries, point clouds, and 3D volumetric fields.

Formulations and chemistry

Compositions, raw materials, concentrations, molecular structure.

STOCHOS also handles images (surface defects, microscopy, inspection), graph and mesh data, and multi-fidelity data that combines cheap simulations with sparse, precise tests.

Built as a library

A Python library, not a black box

STOCHOS ships as a Python library built on PyTorch. It runs on Windows and Linux, on CPU or GPU, and drops into the tools your team already uses, so modeling, optimization, and analysis become code you can automate, version, and reuse, instead of steps locked inside a closed interface.

Fits your stack

Sits alongside NumPy, pandas, PyTorch, Jupyter, and your existing PyAnsys and solver scripts.

Automatable and reproducible

Script parameter sweeps, batch jobs, and validation so results are repeatable and reviewable, not clicked by hand.

Full programmatic access

Call the whole API, extend it, and wrap it in your own tools. No capability is stuck behind a menu.

Yours to run, and to pair

Runs on your own infrastructure, installs offline, and pairs with STOCHOS Flow for a visual, AI-assisted canvas on top.

surrogate.py 
# train an uncertainty-aware surrogate
from stochos.dimgp import dimgp_regr

model = dimgp_regr()
model.fit(X_train, Y_train)          # small, noisy data is fine

# prediction with a confidence interval
Y_pred, Y_lower, Y_upper = model.predict(X_new, CI=0.95)

FAQ

Frequently asked questions

What is a surrogate model?+

A surrogate model is a data-driven approximation of an expensive simulation or experiment. It learns the relationship between inputs and outputs from data you already have, then predicts results for new variants in seconds instead of rerunning the full solve or test each time.

What is uncertainty quantification?+

Uncertainty quantification reports how confident a model is in each prediction. Instead of one point value, you get a value and a measure of its reliability. This lets engineers trust predictions inside the data range and run a full validation only where confidence is low.

What is DIM-GP?+

DIM-GP, a Deep Infinite Mixture of Gaussian Processes, is the core STOCHOS algorithm. It combines neural networks and Gaussian processes into a non-stationary, probabilistic model designed for small, noisy datasets, changing geometries and meshes, and multi-fidelity data, without manual hyperparameter tuning.

What data types does STOCHOS support?+

STOCHOS works with scalar and tabular data, signals and time series, 2D and 3D fields from CFD and FEM, graphs and meshes, geometries and point clouds, images, and multi-fidelity data. This breadth lets one approach cover engineering, chemistry, and formulation problems.
